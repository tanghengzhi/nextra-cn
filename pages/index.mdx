# Nextra

**Nextra** 是一个基于 [Next.js](https://nextjs.org) 的静态网站生成器。

它支持 Markdown 和 React 组件 （[MDX](/features/mdx)）、自动生成的[侧边栏和锚链接](/get-started#sidebar-and-anchor-links)、基于文件系统的路由、内置语法高亮显示、图像优化、自定义布局、多语言以及您喜爱的 Next.js 的所有功能。
