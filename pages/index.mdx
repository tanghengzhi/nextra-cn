# Nextra

**Nextra** 是一个基于 [Next.js](https://nextjs.org) 的静态网站生成工具。

它支持 Markdown 和 React 组件 ([MDX](/features/mdx)), 自动生成[侧边栏和锚链接](/get-started#sidebar-and-anchor-links)，基于文件系统的路由，内置的语法高亮功能，图片优化，自定义布局，多语言，和你最爱的 Next.js 框架的所有特性。
