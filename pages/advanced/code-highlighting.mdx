# 代码高亮

`nextra-theme-docs` 使用 [Prism](https://prismjs.com) 和 [prism-react-renderer](https://github.com/FormidableLabs/prism-react-renderer) 
来高亮显示代码块。本节将介绍如何自定义配置。

## 更多语言

为了保持较小的打包体积，默认仅支持[部分语言](https://github.com/FormidableLabs/prism-react-renderer/blob/master/src/vendor/prism/includeLangs.js)的代码高亮显示。如果你想要支持更多语言, 可以安装以下步骤操作:

1. 运行 `yarn add prismjs prism-react-renderer` 向你的 Nextra 项目添加依赖。
2. 在 `pages/_app.js` 文件里面添加下面代码:

```jsx
import Prism from 'prism-react-renderer/prism'

(typeof global !== "undefined" ? global : window).Prism = Prism

require("prismjs/components/prism-kotlin")
require("prismjs/components/prism-csharp")
```

重新启动应用，就可以支持 Kotlin 和 C# 代码的高亮显示了。
您可以在[这里](https://github.com/PrismJS/prism/tree/master/components)找到支持语言的完整列表。
