# 主题

Nextra 本身基本上可以算是一个插件，它将 Next.js 中的 Markdown 路由规范化为结构数据，并且它不处理任何与样式相关的事情。**主题**才是用来渲染实际页面的，它的工作方式类似于 React 中的布局组件。

Nextra提供了 2 个官方主题：

- [文档主题](/themes/docs)
- [博客主题](/themes/blog)

你还可以扩展自己的主题。这是一个由 [@jaredpalmer](https://github.com/jaredpalmer) 提供的很好的入门实例：

- [Nextra Blank Custom Theme/Boilerplate Example](https://github.com/jaredpalmer/nextra-blank-custom-theme)
